# Entrepreneurial OS — Savant-Level Skill

## Philosophy

Entrepreneurial mastery = **stage-appropriate decisions** · **decision velocity over perfection** · **systems that run without you** · **protecting the asset** (you are the company — burnout is bankruptcy).

## 1. Stage-Gate Model

```
Stage 0 → Exploration    ($0 MRR, pre-idea or pre-validation)
Stage 1 → Validation     ($0–$1K MRR, testing riskiest assumptions)
Stage 2 → Traction       ($1K–$10K MRR, repeatable distribution found)
Stage 3 → Growth         ($10K–$100K MRR, scaling what works)
Stage 4 → Dominance      ($100K+ MRR, building defensible moats)
Stage 5 → Exit / Legacy
```

## 2. Decision Frameworks

### OODA Loop (Founder Edition)
- **Daily OBSERVE:** What's new? Any signal that changes priorities? (15 min)
- **Weekly ORIENT:** Is my mental model still accurate? (30 min) — **Never skip this**
- **Weekly DECIDE:** What's the one experiment this week? (30 min)
- **Daily ACT:** 4h deep work blocks. Ship something.

### Priority Stack (Solo Founder / Life OS)
- **Urgent + Important → DO NOW:** Revenue at risk, paying customer problem, blocking outage
- **Not Urgent + Important → SCHEDULE:** Product, marketing, learning, systems building
- **Urgent + Not Important → AUTOMATE/DELEGATE:** Admin, repetitive, formatting tasks
- **Not Urgent + Not Important → ELIMINATE:** Low-signal scrolling, meetings with no outcome

### Pivot vs. Persist Framework
**PIVOT signals (2+ = consider):** 3 months zero organic growth · churn >8% · pain is "nice to have" · CAC >3x LTV · founder lost conviction

**PERSIST signals (2+ = keep going):** NPS >40 · teeth-gritter retention · one channel <$100 CAC · problem is real and urgent

## 3. Weekly Operating Rhythm

```
Monday (60 min):     Review last week · Set SIP · Block deep work · Schedule customer calls
Tue-Thu (4h/day):    Only SIP work · Phone silent · Email batched
Friday (2h):         Ship something · Review · Record 1 learning, 1 win, 1 mistake
Weekend:             1 day fully off — non-negotiable
```

### Quarterly Strategic Review (90-min solo retreat)
1. Am I in the right stage? Acting accordingly?
2. What is the single most important constraint right now?
3. What should I STOP doing?
4. What would make this quarter 10x better?
5. Is my financial runway healthy?

**Output: 3 quarterly boulders (big rocks, not todos)**

## 4. AI-First Leverage Stack

```
Thinking:     Frontier LLM + Perplexity + NotebookLM
Building:     Claude Code + Cursor
Content:      AI draft → human edit → schedule
Automation:   n8n (self-hosted) + Make.com
Ops:          Cal.com + Lemon Squeezy
```

**Rule:** If you do a task manually more than twice, automate it this week.

## Anti-Patterns

1. **Stage Mismatch:** Building infrastructure before product-market fit
2. **Research Paralysis:** Weeks of research instead of 48-hour experiments
3. **Doing Everything:** Running a job instead of a company — find leverage

## Quality Gates
- [ ] Current stage correctly identified; acting stage-appropriately
- [ ] One SIP defined for this week — not a list
- [ ] Weekly rhythm running: plan Monday, deep work Tue-Thu, ship Friday
- [ ] Pivot/persist decision refreshed within last 90 days
- [ ] Founder energy audit done in last 7 days
